03

Revenue & Token Economy

营收与 Token 经济

现金收入清晰可预期，Token 强化生态留存

💰

主要现金收入

订阅与会员

粉丝 / 学员购买 Neowow 订阅，获得内容 + 工具基础配额

课程与训练营

超创学院的系统课程、集中训练营、认证班

工作室与项目分成

平台撮合的定制项目、短剧、品牌合作，按照工作室 / 平台 / 导师分成

工具与 API 收费

高级模型、算力包、开发者 API 的按量或订阅收费

🪙

Token 经济

价值放大与激励层
1

订阅 → 兑换 Token

订阅与付费统一折算为平台 Token

2

Token 用途

支付工具 / 模型使用、工作室服务、课程升级等

3

供给侧收入

工具制作者和工作室收到 Token，可二次消费或通过系统回收 / 会员间交易变现

4

平台收益

Token 回收折扣差 + 交易手续费 + 订阅与课程现金 = 平台现金流

💡

给投资人的关键信息

  • 现金收入清晰、可预期（订阅 + 课程 + 分成）
  • Token 主要承担生态内部的激励与流转，不依赖投机，强化留存与供给侧积极性
04

Commercialization Roadmap

商业化节奏与里程碑

阶段 + 关键指标，投融资视角下的进度与资金用途

Phase 1

产品与学院起盘

0–12 个月
目标 验证用户需求与商业可行性

关键动作

  • Neowow 平台上线核心功能（订阅、课程、工作室雏形、基础工具）
  • 超创学院首期开班 + 全国巡回公开课若干站
  • 完成第一批学员与导师品牌验证

关键指标

有效订阅人数 首期营收 导师数量 作品数量 留存率
Phase 2

生态成型

12–24 个月
目标 形成「学院 + 平台 + 工作室 + 工具 + Token」的完整闭环

关键动作

  • 完整上线 Token 经济系统
  • 工作室规模与项目数量显著提升
  • 工具与 API 产品化，开始对外输出

关键指标

GMV 工作室收入 Token 年度流转 平台 take rate 现金流健康度
Phase 3

扩张与品牌升级

24 个月后
目标 成为「中国 AI 超创 / AI 导演」领域的标杆品牌与基础设施

关键动作

  • 扩展到更多城市与行业合作（短剧、品牌营销、IP、长内容等）
  • 与大型平台 / 机构的合作项目（联合实验室、定制学院等）

关键指标

年营收规模 全国学员与创作者总量 头部项目与合作案例数
